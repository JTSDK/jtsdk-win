----- IMPORTANT -----

TO RUN JTSDK-{DOC,MSYS,PY,QT} Environment(s):
.. Windows >Start >Programs >JTSDK-XX
.. Launch as you would any other Application
.. Desktop Shortcuts are also provided

TO UPDATE / UPGRADE
.. Start >Programs >JTSDK >Tools >JTSDK-Update
.. Type, update
.. Type, upgrade
.. Follow the prompts, exit when finished.

If you have PROBLEMS with JTSDK, please REPORT them
to the WSJT-Devel List: < wsjt-devel@lists.sourceforge.net >
